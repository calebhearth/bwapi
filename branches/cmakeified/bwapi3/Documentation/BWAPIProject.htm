The BWAPI Project

The Brood War Application Programming Interface (BWAPI) is a free and open source C++ framework for creating AI modules for Starcraft. For more information see the open source homepage.

See the Interfaces and UsingBWAPI articles for a basic understanding of BWAPI.

BWAPI2

The BWAPI2 Interface is not included in this offline documentation. See the BWAPI2 Manual online.
